# Ani2Table

A platform connecting rice farmers directly with consumers.

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000).

### Build for Production
```bash
npm run build
```

## Features

- Landing Page with Hero section
- User Authentication (Login/Signup)
- Admin Dashboard
  - Product Management
  - Analytics & Reports
  - Order Status
  - Transaction & Delivery Management
- Farmer Dashboard
  - Profile Management
  - Product Listings
  - Order Management
  - Messaging
- Consumer Dashboard
  - Browse Products
  - Shopping Cart
  - Order Tracking
  - Messaging

## Technologies

- React 18
- React Router v6
- Tailwind CSS
- Recharts (for analytics)
